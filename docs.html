PoGoAPI Documentation

Introduction

PoGoAPI is a simple RESTful API for retrieving data related to Pokémon Go. The API provides access to information such as evolution chains, move data, and Pokémon statistics.

Usage

Use the provided endpoints to access the necessary data for your Pokémon Go-related applications.

Endpoints
  • Evolution Chain: https://yourdomain.com/evo__.json
  • Move Data: https://yourdomain.com/moves__.json
  • Pokémon Data: https://yourdomain.com/Pokemon__stat__.json
Example Usage

To retrieve the evolution chain of a particular Pokémon, use the provided endpoint and handle the JSON data accordingly.

fetch('https://yourdomain.com/evo__.json')
    .then(response => response.json())
    .then(data => {
        // Handle the data here
    })
    .catch(error => console.error('Error:', error));
